El comando `reword` en un rebase interactivo permite modificar el mensaje de un commit sin cambiar lo que tiene adentro. Git se detiene en ese commit
y abre el editor para que se escriba un mensaje más claro y descriptivo. Es útil para modificar el historial antes de compartirlo.
